Dynamic memory allocation

In c language you can't change size of an array but by using dynamic memory allocation we can fix it.For these there are some functions of it like malloc function is used to takes number of bytes to be allocated as an input and returns a pointer of type void.the syntax for malloc is [ptr = (int*)malloc(30* sizeof (int))].

Calloc and Free

the function of calloc is same as malloc but calloc contains '0' as it's arrays value while malloc contains garbage values and we use (n,) instead of (n*) in calloc syntax.We can use free() function to deallocate the memory. The memory allocated using calloc/malloc is not deallocated automatically.The syntax is [free(ptr);]

Realloc

Sometimes the dynamically allocated memory is insufficient or more than required.Realloc is used to allocate memory of new size using the previous pointer and size.The syntax for Realloc is [ptr = realloc (ptr, 3*sizeof(int));]
